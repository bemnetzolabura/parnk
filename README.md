Step 1 — Download the YouTube Video
text

1. Open browser
2. Go to: https://yt1s.com
   (or https://y2mate.com if first doesn't work)

3. Paste this link in the box:
   https://www.youtube.com/watch?v=UxdPqk_55MY

4. Click "Convert" or "Start"

5. Choose: MP4 — 720p (good quality, smaller size)

6. Click "Download"

7. Save it as: video.mp4
   (Remember where you saved it)
Step 2 — Upload Video to Catbox
text

1. Open browser
2. Go to: https://catbox.moe

3. Scroll down to "File Upload"

4. Click "Choose Files" → select your video.mp4

5. Click the "Upload" button

6. Wait for upload to finish
   (Could take 1-5 minutes depending on file size)

7. You will see a URL like this:
   https://files.catbox.moe/abc123.mp4

8. COPY that URL — you will need it

https://random-name-12345.netlify.app

https://files.catbox.moe/abc123.mp4

COMBINE THEM:
https://random-name-12345.netlify.app?v=https://files.catbox.moe/abc123.mp4

THIS is the link you send to people
